```
(ABCD)
  |    \
C → D   |
  |     |
(CD)  (ABC)
        |    \
      C → A   |
        |     |
       (CA)  (AC)
```

∴ Collection of relations: R1(CD), R2(AC), and R3(BC)

Decomposing D → A:

```
(ABCD)
  |    \
D → A   |
  |     |
(AD)  (BCD)
        |    \
      C → D   |
        |     |
       (CD)  (BC)
```

∴ Collection of relations: R1(AD), R2(CD), and R3(BC)
